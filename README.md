# 希特勒背景人物档案（基于托兰《Adolf Hitler》中译本《从乞丐到元首》）

## 项目说明

本仓库是一份辅助阅读 **约翰·托兰（John Toland）** 《Adolf Hitler: The Definitive Biography》（中译《从乞丐到元首》）的 **结构化中文人物档案集**。围绕希特勒一生的 **76 位** 核心背景人物，按时期整理生平、与希特勒的关系演变、在托兰书中的首次出场、历史评价与延伸阅读。

档案由自动化 agent 分阶段生成：
- **阶段 1（本次已完成）**：大纲、时间线索引、关系图总览、目录骨架。
- **阶段 2（后续批次）**：按每批 10–15 人的节奏，为每位人物生成 800–1500 字的详细档案，并在每批完成后提交。

## 阅读指南

1. 先读 [`大纲.md`](./大纲.md)，获得全部 76 位人物的一览表（按章节时间线排序）。
2. 想按章节阅读，请跳转 [`时间线索引.md`](./时间线索引.md)。
3. 想把握人物网络，请打开 [`关系图/总览.md`](./关系图/总览.md)（内嵌 Mermaid 图，可直接在 GitHub 渲染）。
4. 点击大纲中的人物链接进入对应时期目录，查看详细档案。

## 如何使用这份报告

- **通读者**：按 11 个时期目录顺序阅读，与托兰原书对照。
- **主题研究**：通过 [`大纲.md`](./大纲.md) 的类别标签（家人、NSDAP 早期、党卫队与安全机构、军事将领、外交对手、反对者与刺杀者 等）筛选。
- **交叉引用**：每个人物档案末尾的"延伸阅读"会链接到同仓库的相关人物档案。

## 生成方法与数据源

- 内容以 LLM 对托兰原著及标准德国史研究的知识为主，关键日期与事实会在需要时通过联网核实。
- 每次生成都会通过 GitHub API（blob / tree / commit）批量提交，保留完整 commit 历史。
